Using python packages with ecto projects
========================================

You might want to organize your ecto project as a set of ecto modules, all
accessible to python under a single package.

::

  % ls build/lib/ecto_opencv
  __init__.py
  calib.so
  cv_bp.so
  features2d.so
  highgui.so
  imgproc.so
  lazer.so
  line_mod.so
  projector.so
